**Program Modules & Curriculum Depth**

**Module 1: Python Programming & Software Engineering**

* Python from scratch (variables to OOPs)

* Git, GitHub, Version Control

* Python libraries: NumPy, Pandas, Matplotlib

* Unit testing and debugging

**Module 2: Front-End Development**

* HTML5, CSS3, JavaScript (ES6+)

* React.js fundamentals

* DOM Manipulation & Responsive Design

* UI/UX Design Basics

**Module 3: Back-End Development**

* Python with Django

* REST APIs, Authentication

* Database Integration (PostgreSQL, MySQL)

* MERN Stack Exposure (MongoDB, Express, React, Node.js)

**Module 4: Machine Learning & MLOps**

* Supervised & Unsupervised Learning

* Scikit-learn, Model Evaluation

* End-to-end ML Pipeline Building

* CI/CD for ML, Docker & Kubernetes basics

**Module 5: Deep Learning & Computer Vision**

* Neural Networks with TensorFlow/PyTorch

* Convolutional Neural Networks (CNNs)

* Image Classification, Object Detection

* Real-time deployment

**Module 6: NLP & Language Models**

* Text Preprocessing, Tokenization

* Transformers, BERT, GPT family

* Sentiment Analysis, Chatbots

* Multilingual NLP applications

**Module 7: Generative AI & Large Language Models (LLMs)**

* Open Source LLMs: LLaMA, Mistral, Falcon

* Fine-tuning & Evaluation

* **LangChain** & **LangGraph** for Agent Workflows

* Multi-agent system design

* RAG & LLMOps deployment workflows

**Module 8: DevOps, Cloud, and Deployment**

* Docker, Kubernetes for AI workflows

* AWS, Azure basics

* Model hosting on Hugging Face, FastAPI, Streamlit

* End-to-end deployment of full-stack AI apps

**Module 9: Capstone Projects & Portfolio Building**

* 3 industry-level full-stack AI projects

* LLM-integrated web applications

* Resume & LinkedIn optimization with AI

* Mock interviews, GitHub showcase
